=============
Administrator
=============

Dit gedeelte van de handleiding beschrijft de stappen die nodig zijn om de applicatie correct te configureren.
Wanneer alle vereiste instellingen juist zijn geconfigureerd, verdwijnt de banner bovenaan de applicatie die 
aangeeft dat er configuratiefouten zijn.

Zie :ref:`hier<devops-deploying-configuration>` voor meer informatie.
